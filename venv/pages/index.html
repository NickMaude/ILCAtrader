About

ILCA trader is designed to make the process of buying and selling ILCA/laser sailboats much easier. This website finds laser sailboat postings throughout the internet and displays them here in a simple format. To get started click the 'Find Postings' tab on the left.

My name is Nick Maude and I'm a computer science student and an avid laser sailor. I created this site because I'm tired of spending hours online searching for a used boat or part.

This is an open-source project and the associated GitHub repository can be found here.
